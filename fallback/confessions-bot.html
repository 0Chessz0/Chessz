🔒 Confessions Bot

Anonymous confession system for Discord communities

About

Confessions Bot provides a safe, anonymous way for community members to share their thoughts, secrets, and confessions. With built-in moderation tools and privacy features, it's perfect for creating open dialogue in your Discord server.

✨ Key Features

  • 🤐 Completely anonymous submissions
  • 🛡️ Advanced moderation queue system
  • 📝 Custom confession channel
  • ⚙️ Configurable settings per server
  • 🚫 Built-in spam protection
  • 💬 Reply threads

📋 Commands

  • /confess - Submit an anonymous confession
  • /setup - Configure the bot for your server
  • /premium - Access the moderation queue
